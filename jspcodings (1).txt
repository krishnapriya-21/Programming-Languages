jsp directives
--------------
jsp page directives
-------------------
Page directive is used to provide the instructions to the web container that are specific to the current JSP page. It defines the page specific attributes like scripting language, error page etc.
--------------------
jsp import directive
-------------------
<%@ page import="java.util.Date" %>  
 
<html>
	<head>
		<title>import page directive example</title>
	</head>
	<body> 
		Current date: <%= new Date() %> 
	</body>
</html>
-----------------------------------------------------------------------------------------
session directive
------------------
<%@ page session="true" %> 
 
<html>
	<head>
		<title>session page directive example</title>
	</head>
	<body> 
		<h3>Hello this is a session page directive example.</h3> 
		<%
			out.print("Session id:" + session.getId());
		%>
	</body>
</html>
